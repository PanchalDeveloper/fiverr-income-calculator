# Fiverr Income Calculator

This is a Fiverr Income Calculator web application developed by Aman Panchal. It allows freelancers on Fiverr to calculate their income and choose the best withdrawal method based on conversion rates and fees from PayPal and Payoneer.

## Features

- Calculate income from Fiverr projects
- Compare withdrawal methods (PayPal vs. Payoneer)
- Choose currency for withdrawal
- View calculation history
- Responsive design for mobile and desktop devices

## Technologies Used

- HTML5
- CSS3 (Bootstrap 5.3)
- JavaScript

## How to Use

1. Enter the project amount in USD.
2. Enter conversion rates and fees for PayPal and Payoneer.
3. Choose the currency for withdrawal.
4. Click the "Calculate" button to see the results.
5. Click the "View History" button to see previous calculations.
6. Clear history if needed.

## Installation

1. Clone the repository:

`git clone git@github.com:PanchalDeveloper/fiverr-income-calculator.git`


2. Open `index.html` in your web browser.
